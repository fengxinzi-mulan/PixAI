**设计方案**

该 APP 是一个面向 AI 绘画的本地桌面应用原型，核心定位是“多会话图片生成工作台 + 独立图库管理”。整体设计偏现代化组件风，简约、小清新，主色使用薄荷绿，搭配浅色工作台和柔和状态标签，同时支持白天/黑夜主题切换。

**软件功能**

APP 支持配置图片生成服务，包括 `baseURL`、`apiKey` 和默认模型，默认模型为 `gpt-image-2`。生成接口调用路径为 `${baseURL}/v1/images/generations`，用于提交图片生成任务。

应用采用多会话机制，每个会话拥有独立的 prompt 草稿、模型、图片比例、质量和生成数量。用户可以新建会话、切换会话、编辑提示词、清空草稿，并模拟生成图片。生成结果会显示在当前工作区中，失败记录会以错误卡片形式展示，并保留可查看的错误详情。

图库作为独立页面存在，由顶部“图库”按钮进入。图库集中展示全局历史记录，支持搜索、排序、收藏、删除、参数回填、复制和下载模拟。用户从图库回填参数后，会自动回到生成工作台。

图片结果支持预览、缩放、复制和下载模拟。预览以弹窗方式展示，支持放大、缩小、关闭等操作。

设置区支持显示/隐藏，隐藏后主工作区自动扩展。设置区包含全局服务配置和当前会话参数配置，包括模型、图片比例、质量、生成数量、自动写入历史、失败详情保留等开关。

**界面设计**

界面采用三栏工作台结构：左侧为会话列表，中间为生成工作区，右侧为设置面板。顶部为全局导航栏，包含品牌、当前接口状态、设置开关、图库入口和新建会话按钮。

左侧会话列表已精简，不显示会话标题，只保留状态标签、prompt 摘要和关键参数，减少视觉负担。左下角显示版本号，并提供白天/黑夜主题切换。

中间工作区上方是轻量工具栏，包含“文生图”“自动保存草稿”“已保存”“失败”和“清空”按钮。下方是 prompt 输入框和生成按钮，再往下是图片结果网格。图片卡片使用柔和渐变作为占位视觉，失败卡片使用浅红色状态样式突出异常。

右侧设置区为参数面板，包含“服务配置”和“当前会话参数”。通过顶部“设置”按钮可隐藏或显示，适合用户在专注生成时收起配置区。

图库页面独立于工作台，进入后中间和右侧区域合并为完整图库视图。顶部提供图库标题、说明和返回工作台按钮，下方是全局历史库工具栏和历史卡片网格，适合集中浏览、检索和管理生成记录。
